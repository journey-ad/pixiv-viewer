## Features
- [x] 基础页面
  - [x] 首页信息流
  - [x] 排行榜
  - [x] 作品页面
  - [x] 作者信息页面
  - [x] 设置页面
- [x] 搜索功能
- [x] 以图搜图
- [x] 动图播放
- [x] 动图下载(ZIP/GIF/WebM 仅支持 web 端)
- [ ] 历史记录
- [ ] 左右滑动浏览作品

## Credit
- Vue：前端框架
- Vant UI：组件库
- 自己的 [Pixiv API](https://api.imjad.cn/pixiv_v2.md)：提供大部分接口支持
- [SauceNAO](https://saucenao.com/)：以图搜图功能接口
- [Cloudflare Workers](https://workers.cloudflare.com/)：图像反代服务
- Github Pages：提供页面托管服务
- Travis CI：提供持续集成服务
- HTML5+：App 功能支持
- HBuilder X：云打包 App
